# Domain Prompt — Monopoly Jail & Bankruptcy

## 领域边界

### 监狱系统 — 必须处理
- 入狱触发条件：
  1. 踩到 GO_TO_JAIL 格（索引 21）
  2. 连续掷出三次双数
- 入狱效果：移动到 JAIL_VISIT 格（索引 7），标记为"在狱中"
- 在狱中行为选择（每回合开始时）：
  1. 支付 $50 保释金 → 立即出狱，正常掷骰
  2. 尝试掷双数 → 如果掷出双数则出狱并移动，否则留在狱中
- 第三回合强制：如果已在狱中 3 个回合仍未出狱 → 强制支付 $50 后出狱
- 出狱后：正常参与回合

### 破产系统 — 必须处理
- 触发条件：需要支付金额（租金/税务/保释金）但资金不足
- 破产后果：
  1. 该玩家被标记为"已破产"
  2. 释放该玩家所有地产为"无主"
  3. 该玩家退出游戏（不再参与回合轮转）
- 游戏结束条件：仅剩 1 名未破产玩家 → 该玩家获胜

### 不处理
- 正常的回合流转主循环
- 地产购买/租金计算的具体逻辑
- UI 弹窗的具体设计
- Phase 2 网络相关
