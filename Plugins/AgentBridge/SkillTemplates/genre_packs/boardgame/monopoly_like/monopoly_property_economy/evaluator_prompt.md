# Evaluator Prompt — Property Economy 自检

## 必过检查项

1. **初始资金**：是否为 $1500？起点奖励是否为 $200？
2. **所有权状态**：是否定义了 unowned→owned→released 三种状态及转移条件？
3. **购买规则**：是否只在踩到无主 PROPERTY 格时触发？放弃后是否保持无主（不拍卖）？
4. **租金计算**：基础租金是否来自 tile_data_table？颜色组翻倍条件是否正确（拥有同色全部）？倍率是否为 2？
5. **税务**：是否列出了所有 TAX 格的扣款金额？
6. **资金流转完整性**：是否覆盖了起点奖励、购买、租金、税务四种流转？
7. **资金不足处理**：是否定义了资金不足时触发破产的接口（留给 jail_and_bankruptcy）？
8. **Phase 范围**：是否有拍卖/抵押/建房等 Phase 1 不含的内容？
