# Domain Prompt — Monopoly Property Economy

## 领域边界

### 必须处理
- 初始资金：每人 $1500
- 起点奖励：经过起点 $200（注意：由 turn_flow 触发，但金额定义在此）
- 地产购买：踩到无主 PROPERTY 格时，可选择以标价购买
- 所有权追踪：每个 PROPERTY 格有 owner_player_id（null = 无主）
- 租金收取：踩到他人地产时，支付 base_rent 给地主
- 颜色组加成：拥有同色组全部地产时，租金翻倍（multiplier = 2）
- 税务：踩到 TAX 格时扣除固定金额
- 资金不足检查：支付前检查余额，不足则触发破产（交由 jail_and_bankruptcy 处理）

### Phase 1 不处理
- 地产拍卖（放弃购买后不拍卖，直接保持无主）
- 抵押/解押
- 建造房屋/旅馆
- 交易系统

### 资金流转方向
- 起点 → 银行给玩家 $200
- 购买 → 玩家给银行（标价）
- 租金 → 租客给地主（base_rent 或翻倍后）
- 税务 → 玩家给银行（固定金额）
